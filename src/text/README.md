
# text library

simple text rendering library. <br>

using: [FreeType](https://freetype.org/index.html)



features:

- [x] font loading
- [x] glyph draw
- [x] line draw
- [x] block draw
- [ ] font resize
- [ ] font passed as struct
- [ ] colored text
  - [x] colored glyph
  - [x] colored line
  - [ ] colored block
  - [ ] color coded line
  - [ ] color coded block

- [ ] img / quad draw
- [ ] full utf-8
- [ ] ...
